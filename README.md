## About The Project

This is an AI Powered Fantasy Premier League companion, that helps users make better decisions in fantasy premier league.

Features Include:

- Fantasy Football News
- Predicted Points Table
- Player Season Stats
- Team Fixtures and Squads
- Fixtures/ Results and Match Information
- Team-specific Transfer Advice

## Getting Started

To access the website, go to the website URL (Not available yet)

For developers: to set up and run the application on your local device follow the instructions below:

### Prerequisites

[NodeJS LTS (Node 18 or later)](https://nodejs.org/en/download)

- Dowload node.js with npm (node package manager)

### Installation

`In Terminal/cmd`
1. Clone the repository
`git clone https://github.com/ben-hockley/FPLAICompanion.git`

2. Change Directory into git repository
`cd FPLAICompanion`

3. Install dependencies with NPM (Node package manager)
`npm install`

4. Run the server in development mode
`npm run dev`
